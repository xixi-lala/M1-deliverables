## 一、实验名称
现代数据栈环境搭建与日志脱敏初探
## 二、实验目的
1. 工程认知：掌握轻量级、无依赖、列式执行的 OLAP 数据库（DuckDB）与多线程 DataFrame 工具（Polars）的核心特点。
2. 环境配置：熟练搭建 Python 本地虚拟环境，并成功引入且配置 AI 编程智能体（Agent）以提效研发，适应现代数据工程的工作流。
3. 数据探索：认识千万级模拟业务日志数据集的数据分布、脏数据特征及基础脱敏处理逻辑。
## 三、实验软硬件环境
1. 操作系统：Windows 11
2. 核心运行时：Python 3.13.2、Node.js 20.12.0（LTS版，用于运行Qwen Code）
3. 研发工具：Visual Studio Code（集成终端 + Notebook 插件）、Git
4. 核心依赖库：duckdb、polars、pandas、jupyter、hashlib、faker（AI 自动安装）
## 四、实验步骤与核心操作
### 任务1：隔离与构筑数据工程环境
#### 操作步骤
1. 创建虚拟环境（Python venv）：
```
python -m venv data_env 
.\data_env\Scripts\activate # Windows激活命令
```
2. 安装核心依赖库：
```
pip install duckdb polars pandas jupyter
```
3. 验证环境：
```
python -c "import duckdb, polars as pl, pandas, jupyter; print('环境挂载成功')"
```
